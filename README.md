## Overview

A backend service for a personal journaling app built with Node.js, Express, and TypeScript.

## Features

- User Authentication
- Journal Entry Management
- Categorization
- Data Summary
- Secure Endpoints

## Technologies

- Node.js
- Express
- TypeScript
- MySQL
- Typeorm
